## Käyttöohjeet

#### Ohjelman alustaminen

1. Asenna riippuvuudet komennolla 
```
$poetry install 
´´´

#### Ohjelman ajaminen
```
$ poetry run invoke start
```

#### Testien ajaminen
```
$ poetry run invoke test
```

#### Kattavuusraportin luominen
```
$ poetry run invoke coverage-report
```

#### Pylintin ajaminen
```
$ poetry run invoke lint
```
